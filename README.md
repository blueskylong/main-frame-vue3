# main-frame
只有主框架了登录界面的开发框架

使用了 Vue3  vue-class-component  vue-property-decorator等，编写方式更适合后端开发人员。

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your unit tests
```
npm run test:unit
```

### Lints and fixes files
```

```
